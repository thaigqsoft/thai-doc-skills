# ฟอนต์แห่งชาติ (National Fonts) ที่มากับ thai-docx

โฟลเดอร์นี้รวมฟอนต์ราชการไทยที่ **แจกจ่ายต่อได้ตามสัญญาอนุญาต** เพื่อให้เอกสารที่สร้างด้วย `thai_docx` แสดงผลฟอนต์ได้ถูกต้องทุกเครื่อง

## มีอะไรบ้าง

### `national/` — 10 ตระกูลฟอนต์แห่งชาติ (ลิขสิทธิ์ DIP & SIPA)

| key (ใช้กับ `--font`) | ชื่อฟอนต์ (family) | ลักษณะ |
|---|---|---|
| `sarabun` | TH SarabunPSK | ทางการ มาตรฐานหนังสือราชการ |
| `krub` | TH Krub | อ่านง่าย ทันสมัย |
| `koho` | TH KoHo | หนา เป็นทางการ |
| `niramit` | TH Niramit AS | คลาสสิก |
| `kodchasal` | TH Kodchasal | เป็นกันเอง |
| `baijam` | TH Baijam | ลายมือ |
| `chakrapetch` | TH Chakra Petch | เหลี่ยม เท่ |
| `fahkwang` | TH Fah kwang | โมเดิร์น |
| `k2d` | TH K2D July8 | ลายมือเด็ก |
| `mali` | TH Mali Grade 6 | ลายมือนักเรียน |

แต่ละตระกูลมี 4 น้ำหนัก: ปกติ / ตัวหนา / ตัวเอียง / หนาเอียง (รวม 40 ไฟล์)

### `it9-variant/` — ชุด IT๙ (ตัวเลือกเสริม)

ฟอนต์ TH SarabunPSK และ TH Niramit AS รุ่นที่ปรับให้แสดง **เลขไทยแบบ ๙** สำหรับงานระบบราชการบางระบบ

> ⚠️ **คำเตือน:** ชุด IT๙ ใช้ "ชื่อฟอนต์ภายในเดียวกัน" กับ TH SarabunPSK / TH Niramit AS ปกติ
> ถ้าติดตั้งทั้งสองชุดพร้อมกัน Windows อาจสับสนว่าจะใช้ตัวไหน — แนะนำให้เลือกติดตั้งอย่างใดอย่างหนึ่ง

## ติดตั้งฟอนต์เข้าเครื่อง

```bash
python scripts/install_fonts.py          # ติดตั้ง 10 ฟอนต์แห่งชาติ (per-user ไม่ต้องสิทธิ์ admin)
python scripts/install_fonts.py --it9    # ติดตั้งชุด IT๙ ด้วย
python scripts/install_fonts.py --list   # ดูรายชื่อก่อน
```

หรือบน Windows: เลือกไฟล์ `.ttf` ทั้งหมด → คลิกขวา → **Install** (หรือ Install for all users)

## ลิขสิทธิ์

- โฟลเดอร์ `national/` และ `it9-variant/`: ดูรายละเอียดใน [`LICENSE-NATIONAL-FONTS.txt`](LICENSE-NATIONAL-FONTS.txt)
  สรุป: **ใช้/ทำซ้ำ/ดัดแปลง/แจกจ่ายได้ฟรี** ห้ามขายฟอนต์โดยลำพัง
- ฟอนต์ตระกูล **TH Charmonman / TH Srisakdi / TH Charm of AU** (ลิขสิทธิ์ ศ.ดร.ศรีศักดิ์ จามรมาน, ม.อัสสัมชัญ)
  **ไม่ได้รวมไว้** ในรีโปนี้ เพราะสัญญาให้สิทธิ์แจกจ่ายเฉพาะแก่ DIP/SIPA — หากต้องการโปรดขอจากเจ้าของลิขสิทธิ์โดยตรง

ฟอนต์เหล่านี้เป็นลิขสิทธิ์ของเจ้าของเดิม โครงการนี้เพียงรวบรวมมาแจกจ่ายต่อตามสิทธิ์ที่ได้รับอนุญาตเท่านั้น
